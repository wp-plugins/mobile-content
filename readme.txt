== Description ==

Mobile Content plugin allows you to use shortcodes to display different content, in your posts or pages, depending on the device it is being viewed on. Using these shortcodes you can specify to only show certain content to desktops, tablets, or mobile phones.
See an example <a href="http://dylanfeltus.com/mobile-content/demo/">here</a>.

Shortcodes:<br />
[desktop]Your Desktop Content Here[/desktop]<br />
[tablet]Your Tablet Content Here[/tablet]<br />
[mobile]Your Mobile Content Here[/mobile]

You can see more info about the plugin at:
<a href="http://www.wpmobilecontent.com">www.wpmobilecontent.com</a>

If you have any questions feel free to contact me via my site:
<a href="http://www.dylanfeltus.com">www.dylanfeltus.com</a>

== Installation ==

You can upload the mobile-content.zip file using the Plug-in Installer located at Plugins > Add New in the Wordpress Admin.

or

1. Upload `mobile-content` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the short codes to specify the device.
