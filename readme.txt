== Description ==

Mobile Content plugin allows you to use shortcodes to display different content, in your posts or pages, depending on the device it is being viewed on. Using these shortcodes you can specify to only show certain content to computers, tablets, or mobile phones. You can also target OS like iOS, Android, Blackberry, and Kindle.
See an example <a href="http://dylanfeltus.com/mobile-content/demo/">here</a>.

Shortcodes:<br />
[computer]Content for only computers here[/computer]<br />
[tablet]Content for only tablets here[/tablet]<br />
[mobile]Content for only phones here[/mobile]<br />
[mobile_tablet]Content for phones and tablets here[/mobile_tablet]<br />
[ios]Content for only iOS devices only here[/ios]<br />
[android]Content for only Android devices only here[/android]<br />
[bb]Content for only Blackberry devices here[/bb]<br />
[kindle]Content for only Kindle devices here[/kindle]<br />
Retina Display coming soon!<br />
<br />

== Installation ==

You can upload the mobile-content.zip file using the Plug-in Installer located at Plugins > Add New in the Wordpress Admin.

or

1. Upload `mobile-content` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the short codes to specify the device.
